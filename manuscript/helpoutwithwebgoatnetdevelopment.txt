##  Help out with WebGoat .NET development 

If you are looking for a great project to learn about ASP.NET, C# and Web Application Security, [WebGoat .NET](https://www.owasp.org/index.php/Category:OWASP_WebGoat.NET) is a great place to start.

Please grab the code ([here](https://github.com/OWASP/WebGoat.NET) or [here](https://github.com/OWASP/WebGoat.NET/tree/VS_2010)), read the [current issues](https://github.com/OWASP/WebGoat.NET/issues) and join [the mailing list](https://groups.google.com/a/owasp.org/forum/#!forum/webgoat-.net)

See my [WebGoat.NET in Action (and how I set-it up)](http://blog.diniscruz.com/2013/06/webgoatnet-in-action-and-how-i-set-it-up.html) post if you need help setting it up.

And please fell free to invite others to join the party :)
